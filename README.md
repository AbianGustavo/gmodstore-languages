# Kobralost's gmodstore addons languages
Here, you have all languages files of **[Kobralost's addons](https://www.gmodstore.com/users/kobralost/addons)**.

## Collaboration
If you see a **mistake** or want to **add a new language** to an addon, you must **create a [Pull Request](https://github.com/KB-GModStore/gmodstore-languages/pulls)** with the changes.
If it is accepted, they will be automatically synchronized in the next update.
